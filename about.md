# 소개

AI 시스템을 설계하고 운영하는 엔지니어로 일합니다.
LLM 기반 애플리케이션 개발부터 ML 파이프라인 구축, 데이터 인프라 설계까지 AI 서비스가 실제로 동작하는 데 필요한 모든 것에 관심이 있습니다. 모델을 잘 만드는 것 뿐만 아니라, 시스템 전체를 견고하게 만드는 데에도 관심이 많습니다.

---

## 주요 관심사

- **LLM 엔지니어링**: 프롬프트 설계, RAG 파이프라인, 에이전트 시스템
- **MLOps / LLMOps**: 모델 서빙, 실험 관리, 모니터링, 비용 최적화
- **데이터 엔지니어링**: 파이프라인 자동화, 데이터 품질, 피처 스토어
- **백엔드**: FastAPI, Python, 분산 시스템
